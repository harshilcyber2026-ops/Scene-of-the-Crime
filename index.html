Scene of the Crime

A photo was taken at the scene. The photographer was careless — they left too much metadata behind. The flag is hiding in the image itself.

Description

Every digital photograph carries EXIF metadata — hidden data embedded by the camera at the moment the shot was taken. This includes camera model, software, timestamp, GPS coordinates, and free-text fields like Artist and Copyright.

An operative photographed a restricted scene and uploaded the evidence file without stripping the metadata first. Their operator ID was left embedded in one of the EXIF text fields.

Download the image. Extract its EXIF data. Find the field that contains the flag. No steganography — just metadata readable with any EXIF tool.

// What is EXIF?

EXIF (Exchangeable Image File Format) metadata is automatically embedded in JPEG/TIFF images by cameras and phones.

It contains camera model, timestamps, GPS, settings, and free-text fields like Artist, Copyright, and Description — which can contain anything.
